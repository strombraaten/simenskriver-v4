# simenskriver.no

Personal blog and digital garden at [simenskriver.no](https://simenskriver.no).

Running on [Astro](https://astro.build) with a customized version of the [Astro Modular](https://astro-modular.netlify.app/) theme. Content is authored in [Obsidian](https://obsidian.md) and published via [Vault CMS](https://docs.vaultcms.org/guides/introduction/) — writing in Obsidian and pushing to git triggers an automatic Vercel deploy.
