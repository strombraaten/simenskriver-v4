### Hotkeys

- Open file -> `cmd P`
- minimap
- Console
	- More useful if you're troubleshooting your Sublime package installing or something like that
- Copy and paste history
	- CMD+K, CMD+V
- `Control + Command + G` — Select all occurrences
- Split the interface
	- Click on the tab with command, and then when you want to
- Cmd+shift+P -> Allows you to go anywhere within the command palette
- Find symbols in projects -> Find a particular symbol within all the files of your project
- Open a file in your file system -> `cmd+o`
	- Projects, open a project, when a project is loaded into the sublime interface you can quickly
	- `cmd+p` = fuzzy search to find the file i'm interested in
- `cmd+p` -> `filename:63` will take you to line 63 in that file
- Open multiple windows at once
	- `cmd+p`, hit your right arrow, and then go down to a different file, and hit the right arrow and it'll open that file for you as well, all from that one command.
- cmd+p -> `index#chapter` -> will take you the first instance of that word in that file
- cmd+p -> `#` will show you different parts of the document really quickly
- Find and replace
	- Cmd+f -> brings up the file panel, which you can resize if you want
	- If you want to find all instances of inline, you hit `cmd+f` and type "inline" and that'll show you the first instance of the word inline, and then you'll just jump to the next instance. If you want to go back up you hit shift+enter, instead of enter.
	- `option+enter` will help you find all, so you'd hit `cmd+f` first, and then `option+enter`afterwards, if you'd want to find all files with a certain expression
